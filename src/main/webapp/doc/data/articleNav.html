上一篇下一篇标签

<data name="articleNav" queryLock="" idOrAlias="" ref-article=""/>

可接受的标签属性

属性名 类型 是否必须 默认 说明
idOrAlias string 是 文章id或者别名

结果返回

如果文章上一篇下一篇都不存在，那么返回null，否则返回一个ArticleNav对象

ArticleNav可访问属性如下：

属性名 类型 是否可能为null 说明
previous Article 是 文章，查看文章可供访问属性
next Article 是 文章，查看文章可供访问属性
previous和next不可能同时为null

默认配合模板

<div th:if="${articleNav != null }" class="alert alert-info">
	<p th:if="${articleNav.previous != null}">上一篇:<a th:href="${urls.getUrl(articleNav.previous)}" th:text="${articleNav.previous.title}"></a></p>
	<p th:if="${articleNav.next != null}">下一篇:<a th:href="${urls.getUrl(articleNav.next)}" th:text="${articleNav.next.title}"></a></p>
</div>

文章属性

属性名 类型 是否可能为null 说明
id Integer 否 文章ID
alias String 是 文章别名
title String 否 文章标题
space Space 否 空间，查看空间可访问属性

空间属性

属性名 类型 是否可能为null 说明
id Integer 否 空间ID
alias String 否 空间别名
name String 否 空间名
