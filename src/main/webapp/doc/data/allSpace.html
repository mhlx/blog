所有空间标签

<data name="spaces"/>

结果返回

返回List<Space>，如果没有空间，返回空集合

Space可访问属性如下：

属性名 类型 是否可能为null 说明
id Integer 否 空间ID
alias String 否 空间别名
name String 否 空间名
isPrivate Boolean 否 是否是私有空间
createDate Timestamp 否 空间创建日期
lockId String 是 锁ID，如果没有被锁保护，则为null
isDefault Boolean 否 是否是默认空间
