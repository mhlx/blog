动态列表标签

<data name="newsPage" begin="" end="" queryPrivate="" asc="" pageSize="" currentPage=""/>

可接受的标签属性

属性名 类型 是否必须 默认 说明
begin 日期(关于可以接受的日期格式，请见接受的日期格式) 否 动态撰写日期最小值必须和end属性结合使用
end 日期(关于可以接受的日期格式，请见接受的日期格式) 否 动态撰写日期最大值必须和begin属性结合使用
asc boolean 否 false 正序(时间)
currentPage int 否 1 当前查询页码
pageSize int 否 10 每页展示数目这个数目不会超过设置的最大每页数目

结果返回

无论是否查询到结果，都将会返回一个不为null的PageResult<News>对象，该对象基础可访问属性请见PageResult默认属性，

News可访问属性如下：

属性名 类型 是否可能为null 说明
content String 否 动态内容
write Timestamp 否 动态发布日期
update sql.Timestamp 是 动态最后修改日期
isPrivate Boolean 否 是否是私人动态
comments int 否 动态评论数
allowComment Boolean 否 是否允许评论
