文章归档标签

<data name="archives" mode=""/>

可接受的标签属性

属性名 类型 是否必须 默认 说明
mode enum 否 YMD 归档模式，为Y(按年归档)|YM(按年月归档)|YMD(按年月日归档)其中之一

结果返回

返回一个不为null的ArticleArchiveTree，ArticleArchiveTree可访问属性描述如下:

属性名 属性类型 是否可能为null 说明
nodes List<DateNode> 否 根节点

DateNode可访问属性如下：

属性名 属性类型 是否可能为null 说明
text Message 否 节点本文内容，通过${messages.getMessage(message)}输出
nodes List<Object> 否 子节点，根据归档模式的不同，子节点的类型也不同，在YMD模式下，树被分为4层，前三层为DateNode,最后一层为文章节点, 例如：
2016年(DateNode)
  12月(DateNode)
    10日(DateNode)
      [Article1](文章节点)
在YM模式下，树有三层，前两层为DateNode，第三层为文章节点，例如： 
2016年(DateNode)
  12月(DateNode)
    [Article1](文章节点)
在Y下，树有两层，第一层为DateNode，第二层为文章节点，例如： 
2016年(DateNode)
  [Article1](文章节点)

文章节点可访问属性如下：

属性名 属性类型 是否可能为null 说明
id int 否 文章ID
pubDate Timestamp 否 文章发布日期
lastModifyDate Timestamp 是 文章最后修改日期
title string 否 文章标题
isPrivate boolean 否 是否是私人文章
alias String 是 文章别名
lockId String 是 锁ID
space Space 否 空间
空间可访问属性如下
属性名 属性类型 是否可能为null 说明
alias string 否 空间别名
isPrivate boolean 否 是否是私有空间
lockId String 是 锁ID
