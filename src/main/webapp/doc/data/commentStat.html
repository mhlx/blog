评论统计标签

<data name="commentStatistics"/>

结果返回

返回不为null的CommentStatistics对象

CommentStatistics可访问属性如下：

属性名 类型 是否可能为null 说明
totalArticleComments int 否 文章评论总数
totalPageComments int 否 自定义页面评论总数

默认配合模板

<div class="panel panel-default" th:if="${commentStatistics != null}">
	<div class="panel-heading">
		<h3 class="panel-title">评论统计</h3>
	</div>
	<ul class="list-group">
		<li class="list-group-item"><strong>文章评论数：</strong><span
			th:text="${commentStatistics.totalArticleComments}"></span></li>
		<li class="list-group-item"><strong>页面评论数：</strong><span
			th:text="${commentStatistics.totalUserPageComments}"></span></li>
	</ul>
</div>
