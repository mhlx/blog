随机文章标签

<data name="randomArticle" queryLock=""/>

可接受的标签属性

属性名 类型 是否必须 默认 说明
queryLock Boolean 否 false 是否查询被锁保护的文章

结果返回

返回一个可能为null的Article对象，Article不能直接用于显示，只会保留用于构造访问链接的基本属性

属性名 类型 是否可能为null 说明
id Integer 否 文章ID
alias String 是 文章别名
title String 否 文章标题
space Space 否 空间，查看空间可访问属性

通常配合重定向标签使用，例如: 

<data name="随机文章"/>
<redirect th:if="${randomArticle != null}" th:url="${urls.getUrl(randomArticle)}"/>

空间属性

属性名 类型 是否可能为null 说明
id Integer 否 空间ID
alias String 否 空间别名
name String 否 空间名
