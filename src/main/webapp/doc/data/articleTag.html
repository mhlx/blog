标签集标签

<data name="articleTags"/>

结果返回

返回List<TagCount>，只会为空，不会为null

TagCount可访问属性如下：

属性名 类型 是否可能为null 说明
tag Tag 否 标签，查看标签可访问属性
count int 否 标签关联的文章数量

默认配合模板

<div class="panel panel-default" th:unless="${#lists.isEmpty(articleTags)}">
	<div class="panel-heading">
		<h3 class="panel-title">标签</h3>
	</div>
	<div class="table-responsive">
		<table class="table">
			<thead>
				<tr>
					<th>标签</th>
					<th>博客数量</th>
				</tr>
			</thead>
			<tbody>
				<tr th:each="articleTag : ${articleTags}">
					<td ><a th:href="${urls.getArticlesUrl(articleTag.tag.name)}" th:text="${articleTag.tag.name}"></a></td>
					<td th:text="${articleTag.count}"></td>
				</tr>
			</tbody>
		</table>
	</div>
</div>

标签属性

属性名 类型 是否可能为null 说明
id Integer 否 标签ID
name String 否 标签名
create Timestamp 否 标签创建日期
