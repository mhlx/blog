标签统计标签

<data name="tagStatistics"/>

结果返回

返回不为null的TagStatistics对象

TagStatistics可访问属性如下：

属性名 类型 是否可能为null 说明
articleTagCount int 否 和文章关联的标签数量

默认配合模板

<div class="panel panel-default" th:if="${tagStatistics != null}">
	<div class="panel-heading">
		<h3 class="panel-title">标签统计</h3>
	</div>
	<ul class="list-group">
		<li class="list-group-item"><strong>文章标签数：</strong><span
			th:text="${tagStatistics.articleTagCount}"></span></li>
	</ul>
</div>
