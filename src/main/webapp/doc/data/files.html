文件查询标签

<data name="filePage" extensions="" pageSize="" currentPage="" type="" ignorePaging="" fileName="" path=""/>

可接受的标签属性

属性名 类型 是否必须 默认 说明
path String 否 文件路径，例如path为/a/b/c则查询c目录下的文件
pageSize Integer 否 10 分页数目，不会超过设置的最大分页数目
currentPage int 否 1 当前页码
extensions String 否 文件后缀，如果只需要查询图片文件，那么这个值应该为jpg,jpeg,png,gif(不同后缀用','号分隔)，不填为查询所有
type enum 否 文件类型，DIRECTORY(文件夹),FILE(文件)
fileName String 否 文件名称
ignorePaging boolean 否 false 是否忽视分页，查询全部文件

结果返回

无论是否查询到结果，都将会返回一个不为null的PageResult<BlogFile>对象，该对象基础可访问属性请见PageResult默认属性，

BlogFile可访问属性如下：

属性名 类型 是否可能为null 说明
id Integer 否 文件ID
type enum 否 FILE(文件)|DIRECTORY(文件夹)
path String 否 路径，该路径并不是完整的路径，如果当前指向一个文件，完成路径为/a/b/c/d.jpg，那么这里的path为d.jpg
createDate Timestamp 否 文件创建日期
cf CommonFile 是 如果为文件夹，则为null，查看CommonFile可供访问属性

CommonFile属性

属性名 类型 是否可能为null 说明
id Integer 否 文件ID
size long 否 文件大小(字节)
extension String 否 文件后缀
originalFilename String 否 原始文件名
width Integer 是 宽，图片文件特有属性，其他文件该属性为null
height Integer 是 高，图片文件特有属性，其他文件该属性为null
thumbnailUrl ThumbnailUrl 是 缩略图访问地址，图片文件特有属性，查看ThumnailUrl可供访问属性
url String 否 文件访问地址
downloadUrl String 否 文件下载地址

Thumbnail属性

属性名 类型 是否可能为null 说明
small String 否 小尺寸图片访问地址
middle String 否 中等尺寸图片访问地址
large String 否 大尺寸图片访问地址
