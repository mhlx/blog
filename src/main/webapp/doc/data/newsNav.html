上一条下一条标签

<data name="newsNav" id="" ref-news=""/>

可接受的标签属性

属性名 类型 是否必须 默认 说明
id int 是 动态ID

结果返回

如果文章上一条下一条都不存在，那么返回null，否则返回一个NewsNav对象

NewsNav可访问属性如下：

属性名 类型 是否可能为null 说明
write Timestamp 否 动态发布日期
update sql.Timestamp 是 动态最后修改日期
isPrivate Boolean 否 是否是私人动态
allowComment Boolean 否 是否允许评论
