用户信息标签

<data name="myInfo"/>

结果返回

返回不为空的管理员信息

User可访问属性如下：

属性名 类型 是否可能为null 说明
name String 否 用户名
email String 是 邮箱
gravatar String 是 gravatar头像MD5值，如果邮箱为null，那么这个值也为null

注意

在默认情况下，通过gravatarPrefix可以访问到当前配置的gravatar地址前缀， 然后通过该前缀与邮箱MD5值拼接即可获取头像地址，例如 
<img src="${gravatarPrefix}+${myInfo.gravatar}"/>
