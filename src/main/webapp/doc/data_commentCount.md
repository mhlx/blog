## commentCount

### 完整写法
```
<data name="commentCount" moduleType="" moduleId=""/>
```

### 属性说明
|  属性名称  |说明| 是否必须   | 允许值   | 默认值  |    
|  -  |  -  |  -  |  -  |  -  |
| moduleType   | 评论模块类型   | 是   | string   |
| moduleId   | 评论模块ID   | 是   | string|

### 默认DataName
commentCount

### 返回值
`Integer` 不为null
